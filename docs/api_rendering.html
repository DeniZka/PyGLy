pygly.sorter.sort_plane_back_to_front(render_position, render_direction, objects, object_positions)[source]¶

Sorts objects from back to front along a flat plane.

Sorts objects based upon their dot product value along the camera direction.

Args:
render_position: The position of the camera the scene is being rendered from. render_direction: The direction the camera is facing. objects: List of objects to be sorted. object_positions: List of object positions. These values map directly to the objects list.
Returns:
Returns a sorted list containing the objects.
