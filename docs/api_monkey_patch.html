Monkey Patch¶

Provides methods to replace default behaviors.

pygly.monkey_patch.idle(self)[source]¶

An alternate idle loop than Pyglet’s default.

By default, pyglet calls on_draw after EVERY batch of events which without hooking into, causes ghosting and if we do hook into it, it means we render after every event which is REALLY REALLY BAD http://www.pyglet.org/doc/programming_guide/the_application_event_loop.html

pygly.monkey_patch.patch_idle_loop()[source]¶

Replaces the default Pyglet idle look with the idle() function in this module.
